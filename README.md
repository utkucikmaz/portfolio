## Features

- **Multi-language Support**: 12 Languages including RTL support for Arabic.
- **Theme System**: Seamless dark/light mode switching.
- **3D Interactive Background**: Immersive 3D particle system using Three.js.
- **Animations**: Fluid animations with Framer Motion.
- **Responsive Design**: Optimized for all devices from mobile to 4K.
- **SEO Optimization**: Comprehensive meta tags and structured data.

## Quick Start
### Prerequisites

- **Node.js** >= 20.0.0
- **npm** >= 9.0.0

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/utkucikmaz/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:5173](http://localhost:5173) in your browser.
